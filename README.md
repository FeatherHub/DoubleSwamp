# Double Swamp

사람의 의식에 개입할 수 있는 정령이 신탁을 수행하기 위해 한 사람의 집을 찾아가는 2D 어드벤처 게임입니다.

| 게임 플레이 | 휴대폰 상호작용 |
| :---: | :---: |
| ![Double Swamp 게임 플레이](docs/game_main_screen.png) | ![Double Swamp 휴대폰 상호작용](docs/game_phone_interaction.png) |

## 개발 환경

| 항목 | 내용 |
| --- | --- |
| 엔진 | Cocos2d-x 3.10 |
| 언어 | C++ |
| 대상 플랫폼 | Windows (Win32) |
| 프로젝트 | Visual Studio |

### 빌드 준비

1. [Cocos2d-x 3.10](https://github.com/cocos2d/cocos2d-x/releases/tag/cocos2d-x-3.10)을 내려받습니다.
2. 엔진 폴더에서 `download-deps.py`를 실행합니다.
3. 엔진 폴더 이름을 `cocos2d`로 변경한 뒤 프로젝트 루트에 둡니다.

```text
doubleswamp/
├─ cocos2d/
├─ Classes/
├─ Resources/
└─ proj.win32/
```

## 조작법

| 입력 | 동작 |
| --- | --- |
| 방향키 | 이동 및 메뉴 선택 |
| `Space` | 상호작용 및 선택 |
| `↓` + `Space`를 약 3초간 누르기 | 대상의 의식 안으로 들어가기 |
| `↑` + `Space`를 약 3초간 누르기 | 의식 밖으로 나오기 |
| `Esc` | 휴대폰 화면 등에서 뒤로 가기 |

## 게임 오브젝트

| 오브젝트 | 설명 |
| --- | --- |
| 거울 | 거울을 보려 하면 캐릭터가 고개를 돌리며 시선을 피합니다. |
| 탁상 | 휴대폰이나 TV가 놓이는 가구입니다. |
| 문 | 걸쇠와 돌림쇠로 구성된 잠금장치가 있습니다. 설치 기사가 도착하거나 화재가 발생한 상황에서 상호작용하면 잠금 해제 모드로 전환됩니다. |
| 이불 | 항상 펼쳐진 낡은 이불입니다. 휴대폰 메시지 전송을 마쳤거나 TV 설치 후 TV와 상호작용했다면 잠들어 다음 장면으로 이동할 수 있습니다. |
| 등 | 상호작용할 때마다 켜지고 꺼집니다. 켜도 방은 어둑한 상태를 유지합니다. |
| 벽 | 벽에 붙어 이동하면 이동 속도가 조금 빨라집니다. |
| TV | 설치 기사 방문 이후에는 노이즈 화면을 보여 줍니다. 꿈속에서는 누워 있는 주인공과 빠르게 바뀌는 밤낮이 재생됩니다. |
| 휴대폰 | 알람과 문자 답신, 낙하 및 파손 상태에 따라 상호작용이 달라지는 구형 폴더폰입니다. |
| 지팡이 | 상호작용하면 장착되어 이동 속도가 조금 빨라집니다. 잠들 때 자동으로 해제됩니다. |
